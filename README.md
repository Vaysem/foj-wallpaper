# FOJ-Wallpaper HTML-Шпалери для Windows 11
FOJ-Wallpaper — це застосунок, розроблений для персоналізації робочого столу шляхом встановлення локального HTML-файлу як динамічних шпалер. Він використовує потужність і динаміку сучасних вебтехнологій (CSS, та складні JS-сценарії). Для розширеної взаємодії із системою в застосунку передбачені вбудовані API-доповнення.

## Технічні Особливості
- **Рушій:** FOJ-Wallpaper функціонує на базі вбудованого в операційну систему WebView2 Edge, який використовує рушій Chromium. Це забезпечує повну підтримку всіх можливостей сучасного веббраузера.
- **Оптимізація Ресурсів:** Реалізовано гнучке налаштування програми, що дозволяє *нівелювати системне навантаження* шляхом увімкнення або вимкнення ресурсозатратних API-доповнень. Це дає змогу підібрати оптимальну конфігурацію, особливо для власників ПК із обмеженими обчислювальними ресурсами.
- **Single Instance:** З метою оптимізації та економії обчислювальних ресурсів, застосунок має вбудовану систему **блокування дублюючого запуску**.
- **Drag and Drop:** Додано функціонал **миттєвої зміни шпалер** за допомогою простого перетягування файлу на іконку програми.
- **Підтримувані Формати:** Програма підтримує всі формати файлів, які можуть бути безпосередньо відкриті в стандартному браузері: від **зображень** та **текстових** документів до **відеофайлів** на пряму.
## Безпека та Аудиторія
Політика Безпеки: З міркувань безпеки, у програмі суворо дотримано норм «Політики безпеки». Зокрема, заблоковані AJAX-запити в JS та атрибути src, що посилаються на віддалені ресурси.

Цільова Аудиторія: Цей посібник розрахований на «просунутого» користувача у сфері веброзробки, який має базові знання JS, CSS та HTML. Для використання готових HTML-шпалер ви можете скористатися сайтом розробника або послугами фрилансу.

## API
| Назва | Тип |	Що передається | Короткий опис |
| :--- | :---: | :---: | :--- |
| **parse_url(text)**	 | function | text | Система викликає цю функцію з<br>одним текстовим аргументом, що<br>містить результат парсингу URL. |
| **sys_info({cpu:float,ram:float} )**	 | function | object | Система викликає цю функцію з одним<br>аргументом-об’єктом, що містить дані<br>про відсоток навантаження процесора<br> та оперативної пам’яті. |
| **set_mouse(x:int,y:int)**	 | function | int, int | Система викликає цю функцію,<br> передаючи два аргументи — поточні<br> координати курсора миші (X та Y). |
| **context_command**= {<br>"name command 1":()=>console.log(1),<br>"name command 2":"calc.exe"<br>}	 | function | int, int | Створює контекстне меню (Tray Menu)<br> для лівої кнопки миші (ЛКМ) на основі<br> даних об’єкта. Залежно від типу, він<br>може запустити відповідну функцію в<br> HTML-файлі, або виконати системну<br> команду (якщо надано рядок). |
