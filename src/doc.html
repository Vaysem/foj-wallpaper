Documentation FOJ-Wallpaper

FOJ-Wallpaper is an application designed to personalize your desktop experience by setting a local HTML file as a dynamic wallpaper. It leverages the power and dynamism of modern web technologies (CSS, and complex JS scripts). For extended interaction with the operating system, the application provides built-in API extensions.

Technical Features

  • Engine: FOJ-Wallpaper operates on the WebView2 Edge runtime, which is built into the operating system and powered by the Chromium engine. This ensures full support for all modern web browser features.
  • Resource Optimization: Flexible program settings have been implemented, allowing users to mitigate system load by enabling or disabling resource-intensive API add-ons. This allows for the selection of an optimal configuration, especially for PC owners with limited computing resources.
  • Single Instance: To optimize and conserve computational resources, the application features a built-in system for blocking duplicate application launches.
  • Drag and Drop: Added functionality for instant wallpaper change by simply dragging a file onto the application icon.
  • Supported Formats: The application supports all file formats that can be directly rendered in a standard browser: from images and text documents to video files.

Security and Audience

  • Security Policy: For security reasons, the application enforces strict adherence to the Security Policy. Specifically, tags like AJAX requests in JS, and src attributes that refer to remote resources are all blocked.
  • Target Audience: This guide is intended for "advanced" users in the field of web development who possess basic knowledge of JS, CSS, and HTML. To access ready-made HTML wallpapers, please refer to the developer's website or utilize freelance services.
